parseFloat() Method is used to accept the string and convert it into a floating-point number. If the string does
        not contain a numeral value or If the first character of the string is not a Number then it returns NaN i.e, not
        a number. It actually returns a floating-point number parsed up to that point where it encounters a character
        that is not a Number.


        Use parseInt/parseFloat for the “soft” conversion, which reads a number from a string and then returns the value
        they could read before the error.
      
        For fractions:
        Round using Math.floor, Math.ceil, Math.trunc, Math.round or num.toFixed(precision).
        Make sure to remember there's a loss of precision when working with fractions.
